# 个人知识库网站设计指南

## 设计方法
基于系统的设计方法，灵感源自 GitHub Docs、Dev.to 和 Medium 的阅读体验。专注于可读性、代码呈现和内容层次结构。设计在保持视觉清晰度的同时，优先考虑实用性和信息密度。

## 核心设计原则
1. **内容优先**：通过最佳行长度和间距实现最大可读性
2. **优质代码**：对代码块进行专业语法高亮的高级处理
3. **易扫描结构**：清晰的视觉层次结构，便于快速导航
4. **性能**：快速加载、平滑过渡、极简动画

---

## 排版系统

**主要字体栈：**
- 正文/内容：`Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- 代码：`'JetBrains Mono', 'Fira Code', 'Consolas', monospace`
- 中文支持：确保使用 `'PingFang SC', 'Microsoft YaHei'` 作为合适的回退字体

**字体比例：**
- 文章标题：`text-4xl md:text-5xl font-bold` (36-48px)
- 章节标题 (H2)：`text-3xl font-semibold` (30px)
- 子标题 (H3)：`text-2xl font-semibold` (24px)
- 正文：`text-lg leading-relaxed` (18px，行高 1.75)
- 内联代码：`text-base` (16px)
- 元数据/标签：`text-sm` (14px)

---

## 布局系统

**间距基础单位：** 使用 Tailwind 的 4、6、8、12、16、20 单位
- 组件内边距：`p-4, p-6, p-8`
- 章节间距：`py-12, py-16, py-20`
- 元素间隙：`gap-4, gap-6, gap-8`

**容器策略：**
- 导航栏：全宽，内部最大宽度 `max-w-7xl`
- 文章内容：`max-w-3xl mx-auto` (最佳阅读宽度约 65-75 个字符)
- 代码块：`max-w-4xl` (允许更宽以适配代码)
- 文章网格：`max-w-6xl`

**网格布局：**
- 文章卡片：`grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6`
- 侧边栏 + 内容：`grid-cols-1 lg:grid-cols-4` (1 列侧边栏，3 列内容)

## 组件库

### 导航栏
- 滚动时带有轻微阴影的粘性头部
- 左侧显示 Logo/站点名称
- 醒目的搜索栏，居中
- 右侧显示分类下拉菜单、主题切换器、个人资料/登录按钮
- 高度：`h-16`

### 文章列表页
- 英雄区：带有站点标语和精选文章（包含展示代码/技术主题的大型英雄图片）
- 过滤栏：分类、标签、排序选项
- 文章卡片包含：缩略图、标题、摘要（2 行）、元数据（日期、阅读时间、标签）、悬停提升效果

### 文章详情页
**布局：**
- 面包屑导航
- 文章头部：标题、作者信息、发布日期、阅读时间、标签
- 目录（桌面端固定在侧边栏）
- 主内容区域带有充足的边距
- 相关文章页脚
- 评论区

**Markdown 元素：**
- 标题：渐进式缩小，间距一致
- 段落：段落之间 `mb-6`
- 列表：`ml-6 space-y-2` 并使用自定义项目符号
- 引用块：左侧边框强调，斜体文本，轻微背景色
- 图片：全宽显示，带有标题，懒加载
- 表格：带边框、条纹行、移动端响应式滚动
- 链接：悬停时显示下划线，平滑过渡

### 代码块组件（关键）
**结构：**
```
- 标题栏：左侧语言标签，右侧复制按钮
- 行号列（可选开关）
- 代码内容区域带有水平滚动
- 内边距：p-4 到 p-6
```

**特性：**
- 语法高亮主题：VS Code 深色（默认）、Dracula、Monokai、GitHub 浅色、Nord
- 文章工具栏中的主题切换器
- 行号：`text-gray-500 select-none pr-4`
- 复制按钮：绝对定位，悬停状态显示提示
- 字体大小：`text-sm` (14px)，提高密度
- 水平滚动条（样式与主题匹配）

**待实现的代码颜色主题：**
1. **VS Code 深色**（默认）：深色背景 `#1E1E1E`，暖色调语法颜色
2. **Dracula**：紫/粉强调色方案，背景色 `#282A36`
3. **Monokai**：经典风格，背景色 `#272822`
4. **GitHub 浅色**：浅色模式下的纯白色背景
5. **Nord**：冷蓝灰色调，背景色 `#2E3440`

### 分类/标签 pills
- 全圆角：`rounded-full px-3 py-1`
- 轻微背景色，确保文本对比度
- 悬停：轻微缩放 `hover:scale-105`

### 搜索组件
- 醒目的搜索栏，带有图标
- 即时搜索结果下拉菜单
- 键盘导航支持（↑↓ Enter，Esc）

### 页脚
- 多列布局：关于、分类、最近文章、社交链接
-  newsletter 订阅表单
- 版权信息和技术栈信息

## 响应式行为

**断点：**
- 移动端：单列布局，汉堡菜单，隐藏目录
- 平板设备（md:）：双列文章网格，显示简化导航
- 桌面设备（lg:）：三列网格，固定目录侧边栏，完整导航

**移动端优化：**
- 代码块：将字体缩小为 `text-xs`，确保复制按钮易于触摸
- 表格：水平滚动容器
- 导航：滑动式移动端菜单
- 图片：响应式尺寸 `w-full h-auto`

---

## 图片

**英雄图片：**
- 文章列表首页上的大型全宽英雄图片
- 主题：抽象技术/代码可视化、开发者工作区或数字网格图案
- 为提高文本可读性添加渐变叠加层
- 高度：`h-96 md:h-[500px]`

**文章缩略图：**
- 宽高比：16:9
- 无图片文章的占位符：带有文章图标的渐变
- 对象适配：cover

**文章内图片：**
- 内容容器内全宽显示
- 下方显示较小的静音文本作为标题
- 点击时通过灯箱查看详细视图

---

## 交互

**极简动画：**
- 卡片悬停：`transform hover:scale-102 transition-transform`
- 按钮状态：轻微的透明度/背景变化
- 代码复制：简短的成功对勾动画
- 主题切换：平滑的颜色过渡 `transition-colors duration-300`

**禁止动画的场景：**
- 页面导航
- 文本渲染
- 滚动触发效果（保持快速）
